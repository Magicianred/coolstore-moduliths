# coolstore-moduliths

The coolstore website on modular monolith

## How to run

```bash
$ docker-compose -f docker-compose.yml -f docker-compose.override.yml up
```

```bash
$ powershell -File migrations.ps1
```

Start `CoolStore.Api` and `CoolStore.UI.Blazor`
